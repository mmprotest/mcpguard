# Security Policy

## Reporting a Vulnerability

Please report security issues to [security@example.com](mailto:security@example.com). Include a detailed description, steps to reproduce, and any relevant logs. We aim to acknowledge reports within 2 business days.

For sensitive disclosures, you may use our PGP key (available upon request) to encrypt communications.

## Supported Versions

Only the latest minor release of `mcpguard` receives security fixes.
